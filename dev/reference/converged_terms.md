# Converged Terms

Tests whether each term of an object has converged.

## Usage

``` r
converged_terms(x, ...)
```

## Arguments

- x:

  An object.

- ...:

  Other arguments passed to methods.

## Value

A list of uniquely named logical objects with whether each term has
converged.
