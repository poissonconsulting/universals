# Parameter Dimensions

Gets the dimensions of each parameter of an object.

## Usage

``` r
pdims(x, ...)
```

## Arguments

- x:

  An object.

- ...:

  Other arguments passed to methods.

## Value

A named list of integer vectors of the dimensions of each parameter.
