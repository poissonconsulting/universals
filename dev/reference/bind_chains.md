# Bind by Chains.

Binds two MCMC objects (with the same parameters and iterations) by
chains.

## Usage

``` r
bind_chains(x, x2, ...)
```

## Arguments

- x:

  An object.

- x2:

  A second object.

- ...:

  Other arguments passed to methods.

## Value

The combined object.
