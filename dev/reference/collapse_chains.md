# Collapse Chains

Collapses an MCMC object's chains into a single chain.

## Usage

``` r
collapse_chains(x, ...)

# Default S3 method
collapse_chains(x, ...)
```

## Arguments

- x:

  An object.

- ...:

  Other arguments passed to methods.

## Value

The modified object with one chain.
